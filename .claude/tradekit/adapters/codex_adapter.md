# Adapter — Codex

How to run TradeKit inside Codex / Codex-in-VS-Code.

- Use the same command cards; Codex is well suited to `/tk slice` → `/tk implement` → `/tk verify`
  focused, verification-ready slices.
- Keep one ticket at a time; verify before commit.
- Respect each card's `write_access`; do not broaden scope.
