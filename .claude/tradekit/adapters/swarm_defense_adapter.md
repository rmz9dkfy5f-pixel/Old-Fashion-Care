# Adapter — Swarm Defense

How to apply TradeKit to the Swarm Defense project (or any game/app repo).

- Start with `/tk scan` to capture branch, stack, scripts, and current snapshot.
- Use `/tk decide` to pick the next safe gameplay/feature slice.
- Keep `/tk implement` to one ticket; run `/tk verify` (build + run + smoke test) before handoff.
- Never broaden scope into unrelated systems.
